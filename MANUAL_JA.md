# 🎀 ぷろんぷたん 使い方マニュアル

---

## ぷろんぷたんってなに？

**AIとのチャットで描かれた「情景」を、自動でイラストにして見せてくれる “AIチャットのお供”** だよ。
会話やロールプレイのなかで「いま、こういう場面」とAIが描写してくれたら、その場面をそのまま Stable Diffusion（Forge）が生成 — 読みながら、絵で見られる。

仕組みはシンプル。AIに専用フォーマット（「📋 お約束」）でプロンプトを書いてもらうと、ブラウザの監視スクリプトがそれを検知 → ぷろんぷたんが受け取って SD へ送り、自動で生成。
**「AIにプロンプト書いてもらう → コピペ → SDに貼る」の手間はゼロ。** チャットに集中したまま、情景がどんどん絵になっていくよ。

※ もともとは「AIチャットの場面を、画像としても出せたらいいな」という発想から生まれた “AIチャット支援ツール” だよ。いまは Claude・ChatGPT・Gemini・Grok・Copilot などのチャット画面に対応してるよ。

---

## 起動方法

1. `起動する.bat` をダブルクリック
2. アプリが起動したら、右上のポートが `🎀 ポート 787x` に変わるのを確認
3. Stable Diffusion（Forge）も起動しておいてね。左上のランプが 🟢 になれば接続OK！

## Tampermonkey スクリプトの準備

アプリを使うには、ブラウザに2つのスクリプトを入れておく必要があるよ。

### sd_monitor.user.js（必須）
AIチャットのページを監視して、プロンプトを自動検知・送信するスクリプト。

**対応AIサービス**
- Claude (claude.ai)
- ChatGPT (chatgpt.com)
- Gemini
- Grok
- Copilot

> 💡 **OpenWebUI などローカルのチャットUIで使うとき**は、`sd_monitor.user.js` の先頭にある `@match` の並びに自分のURL（例：`// @match http://localhost:8080/*`）を1行追加してね。

**インストール方法**
1. Tampermonkey の管理画面を開く
2. 「新規スクリプト」を作成
3. `sd_monitor.user.js` の内容をまるごと貼り付けて保存

### forge_sync.user.js（任意）
Forge のページとぷろんぷたんを双方向につなぐスクリプト。

- **📤 設定を送る** … Forge の生成設定（Steps・CFG・解像度・サンプラー）をワンクリックでぷろんぷたんに同期
- **📥 取り込む** … ぷろんぷたんが受信したポジ・ネガを、Forge のプロンプト欄にワンクリックで入力
- **🪄 自動入力** … トグルONにしておくと、新しいプロンプトが届くたびに Forge の欄へ自動で反映（初期値ON）

**インストール方法**（sd_monitor と同様）  
インストール後、Forge のページを開くと右下にボタンが並ぶよ。

## AIへのお約束

AIにプロンプトを書いてもらうとき、専用のフォーマットを使う必要があるよ。  
**「📋 お約束」ボタン**を押すと、お約束の文章がクリップボードにコピーされるから、そのままAIに貼り付けてね！

### プロンプトのフォーマット

ポジティブは 🔴 で始めて 🟥 で終わる。ネガティブは 🔵 で始めて 🟦 で終わる。

```
🔴
masterpiece, best quality, 1girl, ...
🟥

🔵
worst quality, low quality, bad anatomy, ...
🟦
```

## 画面の見方

![ぷろんぷたんのメイン画面](docs/img/screenshot_main.png)

### ステータスカード（右上）
| 表示 | 意味 |
|---|---|
| ✓ SD接続中（緑） | SDに接続できてるよ |
| ○ SD未接続（赤） | SDが見つからない（起動・URLを確認してね） |
| ⚡ 自動 | プロンプト受信したら即生成 |
| 👀 確認 | 手動でボタンを押して生成 |
| ✨ 生成中（明滅） | いま絵を作ってる合図。チップがふわふわ光るよ |
| タイムスタンプ | P=ポジ受信時刻 N=ネガ受信時刻 ✨=生成完了時刻 |

### ポジ・ネガボックス

欄の横の小さなピルが、いまの状態を教えてくれるよ。

| ピル | 意味 |
|---|---|
| ○ 待機中（灰） | まだ何もない |
| ✓ 受信完了（緑/赤） | プロンプトが届いた（届いた瞬間につくよ） |
| ✓ 手入力 | 自分で入力・編集した内容が入ってる |
| ✓ 生成済み（紫） | この内容はもう絵になった（新しく届くとまた受信完了に戻るよ） |

欄は自由に編集できるよ！詳しくは次の「手動入力での生成」へ。

### プレビューと履歴

- プレビューの大きさは **S / M / L** で切替。**Hide** を押すと絵・履歴・ファイル名をまとめて非表示にできるよ（とっさのプライバシー用🙈）
- 下の「最近作った絵」は履歴カルーセル。**‹ ›** で1枚ずつ送れて、クリックすると大きく表示
- 生成が終わると、**ピンクの選択枠が自動で最新の絵に移動**するよ

## ✍️ 手動入力での生成

ポジ・ネガ欄は**直接入力もOK**！

- 欄に書く・貼り付ける → 生成ボタンが点灯 → 押せば生成
- **受信したプロンプトの「ちょい直し」もそのまま反映**されるよ（直すとピルが「手入力」に変わる）
- スパイスの紫色の追記表示が残っているときは、欄を触ると自動で消えてから編集モードになるよ（スパイスの二重掛け防止。スパイスは生成時にちゃんと合成されるから安心してね）

---

## ✨ ポジスパイス

生成時にポジティブプロンプトへ自動で追記する「おまけ」設定だよ。

1. ⚙️（歯車）→ スパイス内容欄（1〜9）に好きなフレーズを入力して保存
2. メイン画面のピル（1〜9）を押してON（紫色）にすると追記される
3. 複数選んでも大丈夫！全部カンマ区切りで追加されるよ

---

## 💪 ネガ補強

手・指・足の解剖学的破綻をネガに自動追記するスイッチ。  
右パネルの「💪 ネガ補強（手・指・足）」のチェックボックスをONにするだけ。  
内容は ⚙️ 設定から自由に編集できるよ。

## 生成モード

| モード | 動作 |
|---|---|
| ⚡ 自動 | ポジ＋ネガが届いたら即生成。ポジだけのときは5秒待ってから生成 |
| 👀 確認 | プロンプトが届いてもボタンを押すまで生成しない |

---

## 📌 ミニモード

右上の **Mini** ボタンで、ポラロイド風の小窓に変身！チャットの隅に置いて、絵が届くのを眺めながら会話できるよ。

![ミニモード（ポラロイド風）](docs/img/screenshot_mini.png)

- 常に最前面・絵が主役（タイトルバーやパネルは全部隠れる）
- 窓のサイズを変えると絵もぴったり追従（位置とサイズは記憶されるよ）
- 戻るときは右下の ↗ ボタン

### ミニのランプの見方

| ランプ | 意味 |
|---|---|
| P / N がペカペカッと点滅→点灯 | 新しいプロンプトが届いた！ |
| GEN が橙色でふわふわ明滅 | いま生成中 |
| GEN が緑のチェック | 生成完了 |
| P / N が薄いチェックに戻る | この受信ぶんは絵にできたよ、の合図（次が届くとまた点灯） |

※ ミニ中は自動生成モードに固定されるよ。

## ⚙️ 設定

右上の歯車マークから開けるよ。

| 項目 | 説明 |
|---|---|
| SD接続設定 | Stable Diffusion の URL（デフォルト: http://127.0.0.1:7860） |
| Forge設定 | Forgeから設定を取り込むボタン |

| ネガ補強テンプレート | 補強で追記されるネガの内容 |

## よくある質問

**Q. プロンプトが届かない**  
→ sd_monitor.user.js がインストールされているか確認。ぷろんぷたんが起動しているか確認。

**Q. SDに繋がらない**  
→ Stable Diffusion（Forge）が起動しているか確認。設定のURLが正しいか確認。

**Q. 同じプロンプトを送っても生成されない**  
→ 仕様です。同じ内容の連続自動生成を防ぐため、オレンジ色で「同じプロンプトだからスキップ！（ボタンで作れるよ）」と案内が出ます。**生成ボタンを押せば何度でも作れます**（シードは毎回ランダムだから違う絵になるよ）。

**Q. 生成中に新しいプロンプトが届いた**  
→ 自動モードの場合、現在の生成が終わったあとに自動で次の生成が始まります。
